# 🐼 DocPo — Google Docs Proposal Formatter

DocPo is a Google Docs-based document formatting and writing assistant designed for structured proposal and research documents.

It combines automated formatting with AI-assisted content generation to help create professional, well-structured documents directly inside Google Docs.

---

## 🚀 Features

### 📄 Document Formatting
- Structured heading formatting (Chapter / Section based)
- Table formatting and normalization
- Image cleanup and alignment
- Footnote processing and placement
- Paragraph and layout standardization

---

### 🤖 DocPo AI (Built-in Assistant)

DocPo includes an integrated AI assistant that helps:

- Generate full proposal sections
- Expand or rewrite selected content
- Maintain formal, institutional tone
- Insert structured citations using placeholders
- Follow strict document formatting rules automatically

The AI is designed specifically for:
- Government proposals  
- Research documents  
- RFP responses  
- Policy-style writing  

---

### 🧩 Sidebar Interface

- Clean sidebar UI inside Google Docs
- One-click document formatting
- Integrated AI prompt access
- Visual feedback during formatting

## 🧠 How It Works

DocPo runs inside Google Docs using Apps Script.

- Adds a custom menu: `🐼DocPo`
- Opens a sidebar interface
- Formats document structure automatically
- Uses AI prompts to generate structured content
